# himo-ui-h5
基于[WebComponents](https://stenciljs.com/docs/introduction) 能力去设计组件,做到与技术栈无关，开箱即用，告别重复轮子

- 在任何平台或设备上工作        
- 适用于任何前端框架，因此可以轻松地跨使用不同技术堆栈的团队和项目使用      
- 体积小
